# azurehakua.moe

This is a simple static website built using a template called Ethereal by HTML5 UP. It includes a responsive layout, easy customization, and fast loading times.

## Features

- Responsive design
- Easy to customize
- Fast loading times

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. Customize the content to fit your needs

## Customization

To customize this website:

1. Edit the HTML files in the root directory
2. Modify the CSS in the `css` folder
3. Update images in the `images` folder
